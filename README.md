# Knowy

Knowy es un portal de compra y venta de cursos online, diseñado para conectar instructores y estudiantes en una plataforma moderna, rápida y segura.

## Tecnologías utilizadas

- **Next.js**: Framework de React para aplicaciones web modernas y optimizadas.
- **TailwindCSS**: Framework de CSS utilitario para un diseño rápido y responsivo.
- **Strapi.js**: Headless CMS para la gestión de contenido flexible y potente.
- **Express.js**: Framework minimalista para construir el backend y las APIs.
- **Appwrite**: Plataforma backend como servicio para autenticación, base de datos y almacenamiento seguro.

## Características principales

- Registro y autenticación de usuarios.
- Publicación y compra de cursos online.
- Panel de administración para instructores.
- Gestión de contenido y usuarios.
- Interfaz moderna y responsiva.
